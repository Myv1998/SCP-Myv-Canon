# SCP-Myv-Canon
A comprehensive organizational framework for the SCP universe. It includes governance hierarchy, departmental structure, security clearance levels, and intelligence agencies documentation. All materials presented as in-universe classified documents formatted in LaTeX.

## Documents

- **ADM-4072-ORGSTRUC** — SCP Foundation Organizational Structure and Governance Overview
- **ADM-0223-CLRLVL** — SCP Foundation Security Clearance Levels
- **FIA-0091-ORGSTRUC** — Foundation Intelligence Agency Structure, Ranks, and Hierarchy
- **UNGOC/HICOM/ADM-3091** — United Nations Global Occult Coalition Organizational Structure

## Additional Files

- `scp_foundation_org_tree.html` — A short interactive visual org chart, viewable in any browser.
